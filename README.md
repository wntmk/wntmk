# Hi there! I'm　Tomoaki Iwano

立命館大学大学院に在籍し、VR空間における音声コミュニケーションの研究・開発を行っています。
フロントエンド（VR）からバックエンド、LLMを活用したアプリケーションまで幅広く興味を持ち、開発に取り組んでいます。

## 主な開発実績

### 1. 多言語音声翻訳コミュニケーションVRアプリ（研究用プロダクト）
Meta Quest 3向けに、リアルタイムな多言語音声会話を可能にするVR環境を構築しました。
- **開発環境:** Unity / C# (Android向けビルド)
- **ネットワーク・音声通信:** Photon Voice 2
- **音声処理基盤:** Google Cloud API (STT / Translation / TTS) を統合した処理パイプライン
※3Dアセット等の容量が非常に大きいプロダクトのため、本アカウント（GitHub）ではなくローカル環境にてバージョン管理・開発を行いました。

### 2. LLMを活用したWebチャットボットアプリ
大規模言語モデル（LLM）のAPIを活用し、リアルタイムに対話が可能なWebアプリケーションを開発しました。
- **言語・フレームワーク:** Python / Flask
- **主な技術:** LLM APIの統合、WebベースのUI/UX設計、バックエンド処理の実装
